gray code

格雷码（gray code/reflected binary code），循环二进制单位距离码，是任意两个相邻编码只有一位差异的编码，属于可靠性编码。

一个简单的例子，普通二进制中011->100(3->4)时，三个bit位都需要取反，因此在三位未完全转换时，存在6种中间状态（之转换了一位+剩一位没转换），但是格雷码中010->110只需要一位取反，避免了其他的可能。
