redis--通用指令

针对redis中的key的相关应用。

Key通用指令

基础操作

  1. EXISTS key [key ...]
  2. DEL key [key ...]
  3. TYPE key

时效性控制

  1. EXPIRE key seconds
  2. PEXPIRE key milliseconds
  3. EXPIREAT key timestamp
  4. PEXPIREAT key milliseconds-timestamp
  5. TTL key
    1. 当key不存在(过期)返回-2
    2. key未设置失效时间返回-1
  6. PTTL key
  7. PERSIST key

查询模式

  1. KEYS pattern

更多

  1. help @generic

数据库通用指令

基本操作

  1. SELECT index
  2. PING [message]
  3. QUIT -
  4. ECHO message
  5. help @connection

数据移动

  1. MOVE key db

数据清除

  1. FLUSHDB
  2. FLUSHALL
  3. DBSIZE
  4. help @server
